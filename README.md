## ✨ 새로운 기능

### 1. 프리미엄 카드 디자인
- SVG 기반 고유 캐릭터 일러스트
- 등급별 시각 효과 (Common, Rare, Epic, Legendary)
- 레전더리 카드 빛나는 애니메이션

### 2. 방 만들기 시스템
- 방 생성 (6자리 코드)
- 방 입장 (코드 입력)
- 친구와 대전!
